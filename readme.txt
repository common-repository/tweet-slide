This is tweet slide plugin. You can use this plugin to show twitter tweet in sliding and fade effect.


== Description ==

You are bored to see the static twitter tweet. We, ThemeWar team bring you an AWESOME plugin for showing your tweet in sliding and fade effect. This is a full featured plugin, Comes with lots of options. Here is the features list of the plugin:

1. Parallax & Non Parallax
2. Custom Background Image
3. Custom Background Color
4. Slide & Fade Effect
5. Slider Controller (Bullet & Arrow)
6. Font Size & Color Manageable
7. Show Hide Profile Picture
8. Show Hide Screen Name
9. Show Hide Tweet Time
10. 100% Responsive

If you want to see demo check it: http://themewar.com/plugins/tweetslide/

Rate it if you like it.


== Installation ==


This is pretty simple. Follow the step below. Install through wordpress.org

1. Visit 'Plugins > Add New'
2. Search for 'tweetslide'
3. Activate tweetslide from your Plugins page. (You will be greeted with a Welcome page.)

Manual install:

1. Upload `tweetslide.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
